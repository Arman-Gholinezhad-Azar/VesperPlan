# VesperPlan

**A beautiful, local, offline-first Planning Application** that combines Todo List and Calendar with powerful Time Blocking.

Built with pure Python and Domain-Driven Design (DDD) — no frameworks, no database, no accounts, no cloud.

---

## Philosophy

Most planning apps try to do everything and end up overwhelming you.  
VesperPlan is intentionally simple: **Focus on what matters — your tasks and your time.**

You can:
- Add tasks to a clean todo list
- Drag them into your calendar as time blocks
- Or work directly in the calendar without using the todo list
- Clear everything when you want a fresh start

---

## Features

- **Local First** — Everything saved on your computer (JSON files)
- **Todo List + Calendar** combined in one app
- **Time Blocking** — Assign tasks to specific time slots
- **Direct Calendar Editing** — Add, rename, delete, move time blocks without todo list
- **Clean & Minimal UI** (using Tkinter)
- **Domain-Driven Design** architecture (clean, maintainable, scalable)
- **No Login, No Tracking, No Ads**
- **Data Control** — One-click clear full todo list or full calendar

---

## Tech Stack

- **Python 3.11+**
- Pure Python (no heavy frameworks)
- Domain-Driven Design (DDD)
- JSON for persistence
- Tkinter (for GUI)
